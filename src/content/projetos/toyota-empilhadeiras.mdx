## Introdução

Estabelecida no Brasil desde 2004, a Toyota Empilhadeiras tem como missão ser a empresa de maior credibilidade no seu segmento. Sempre trazendo o cliente em primeiro lugar, a Toyota percebeu a necessidade de modernizar o seu site para facilitar a navegação dos usuários.

Para isso, a [Agência Chleba](https://www.chleba.net/) foi contratada, onde fui o responsável por desenvolver uma interface que transmitisse a solidez e confiabilidade do nome Toyota.

## Solução

Para atender os requisitos solicitados, foi definido que a interface deveria manter a identidade Toyota utilizando formas retangulares para passar segurança, as imagens complementariam a interface de forma uniforme.
